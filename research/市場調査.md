## 1. 企業研修市場（B2B）

### 1.1 国内の企業研修市場規模（推定）

国内の企業向け人材育成・研修市場は、2025年時点で**推定1兆2,000億〜1兆5,000億円規模**と推計される。

| 区分 | 推定規模 | 備考 |
|------|----------|------|
| 企業内教育費全体（研修費・教育訓練費） | 約1兆2,000億〜1兆5,000億円／年 | 企業の人件費の推定1〜2%相当 |
| うち外注・アウトソーシング市場 | 約3,000億〜5,000億円／年 | 全体の25〜35%が外注化と推定 |
| 研修サービス専業市場（研修会社収益合計） | 約2,000億〜3,500億円／年 | eラーニング含む |

> 出典参考：矢野経済研究所「人材育成・研修サービス市場」、リクルートワークス研究所各種報告（いずれも概算・推計）

**背景**

- 日本企業全体（約400万社）のうち、組織的な研修投資を行う企業は従業員50名以上の中堅・大企業が中心
- 従業員1,000名以上の大企業（約6,000社）が研修費全体の約60%を占めると推定
- 中堅企業（50〜999名、約20万社）は1社あたり年間100万〜2,000万円規模の研修費用を投下

### 1.2 研修外注市場の成長率・トレンド

**市場成長率（推定）**

| 期間 | 成長率（推定） | 主な成長要因 |
|------|--------------|-------------|
| 2020〜2022年 | 一時的に縮小（▲5〜10%） | コロナ禍による対面研修の停止 |
| 2022〜2024年 | 回復・拡大（+8〜12%／年） | オンライン研修の普及・リスキリング需要 |
| 2025〜2027年（予測） | 安定成長（+5〜8%／年） | DX人材育成・法令対応研修の継続需要 |

**注目トレンド**

1. **オンライン・ハイブリッド研修の定着**
   - コロナ禍を契機にオンライン研修が急速に普及
   - 2024年時点で企業研修の推定50〜60%がオンラインまたはハイブリッド形式
   - 地方企業・多拠点企業でのアクセシビリティ向上が追い風

2. **リスキリング関連予算の拡大**
   - 政府の「リスキリング支援」施策（2022〜）により企業の研修予算増加傾向
   - DX・AI活用スキル研修への投資が急増（推定年率20%超の伸び）

3. **研修のマイクロラーニング化**
   - 長時間集合研修から短尺・反復型の学習スタイルへの移行
   - スマートフォン対応コンテンツの需要増加

4. **研修効果測定（ROI可視化）への関心**
   - ラーニングアナリティクスの需要増加
   - 単なる研修実施から「効果・行動変容の証明」へのシフト

### 1.4 外注化が進む背景・ドライバー

**1. 人事部門の人員不足・専門性の限界**

- 多くの中堅企業では人事・教育担当が1〜3名程度（兼任含む）
- 研修コンテンツ設計・ファシリテーションには専門スキルが必要
- 最新の学習設計手法（インストラクショナルデザイン等）は内製が困難

**2. コスト構造の合理化**

- 内製化すると研修担当者の固定費（年収500〜700万円）がかかる
- 外注ならスポット費用のみ発生し、ROI比較で外注が有利になるケースが多い
- 助成金（人材開発支援助成金等）を活用することで実質負担が軽減

**3. テーマの多様化・高度化**

- DX、AI、グローバル対応など専門性の高いテーマは内製が困難
- 変化のスピードが速く、社内ナレッジのアップデートが追いつかない

**4. 若手定着・離職防止の課題深刻化**

- 新卒入社3年以内の離職率：推定30〜35%（厚生労働省データ参考）
- 「成長できない」「学べない」ことが離職理由の上位に
- 体系的な育成プログラムを求める企業ニーズが高まっている

**5. 法令・社会的要請への対応**

- ハラスメント対策義務化（2020〜）、障害者差別解消法改正、育児介護休業法改正など
- コンプライアンス研修のアップデートを継続的に行う必要があり、専門外注が合理的

### 2.2 リスキリング需要の拡大

**政策・制度的背景**

- 2022年：岸田政権が「リスキリングへの投資5年間で1兆円」を表明
- 2023〜2024年：「教育訓練給付金」の拡充、個人への直接補助スキームの整備
- 2025〜：AI・DX人材育成を国策として推進、民間投資も活発化

**リスキリング需要の実態（推定）**

| 指標 | データ（推定・参考） |
|------|-------------------|
| 「スキルアップしたい」と思う社会人比率 | 約70〜80%（各種意識調査より概算） |
| 実際に学習行動に移している比率 | 約30〜40%（意欲と行動のギャップ大） |
| 学習に月3,000円以上支出する割合（20〜30代） | 約20〜25% |
| 転職のためにスキルアップを目指す若手比率 | 約45〜55%（転職市場の活性化と連動） |

**リスキリングの主要テーマ（個人需要）**

1. データ分析・Excel/Python/SQL（DX対応）
2. マーケティング・デジタルマーケティング
3. プロジェクトマネジメント・業務改善
4. ビジネスコミュニケーション・プレゼン
5. キャリアデザイン・自己分析

### 2.4 成長ドライバー

#### DX・AI化による「スキルの陳腐化」不安

- ChatGPT等の生成AI普及により「自分の仕事がなくなるかもしれない」という不安が顕在化
- AIに代替されないスキル（創造性・対人スキル・業務設計力）への需要増
- DXスキル習得の圧力が企業・個人双方にかかっている

#### 転職市場の活性化

- 2020年代に入り「終身雇用神話の崩壊」が若手層に浸透
- 転職回数の許容度が上がり、スキルポータビリティへの意識が高まる
- 転職に向けたスキルアップ投資が個人消費として定着しつつある

#### 副業・フリーランス市場の拡大

- 「副業解禁」の流れにより、本業以外の収入源を作ろうとする若手増加
- 副業に必要なスキル（営業・制作・コンサルなど）習得のための学習投資
- フリーランス人口：推定約300万人（2025年推計）、増加傾向

#### 学習コンテンツのコモディティ化と差別化需要

- 動画コンテンツ自体は低価格化・無料化が進み、情報は「タダ」に近い
- 「情報を得る」より「行動変容・習慣化・コミュニティ」に価値がシフト
- 伴走型・アウトプット重視・コミュニティ型の学習サービスへの需要増

## 3. 市場の課題・機会

### 3.1 既存市場の非効率・アンメットニーズ

#### B2B（企業研修）の課題

| 課題 | 現状 | 生じる問題 |
|------|------|-----------|
| 研修効果の不透明さ | 「やりました」で終わる研修が多い | 予算承認が取りにくい、継続発注されない |
| 単発・点の施策 | 年1回の集合研修で終了 | 学習定着率が低く、行動変容が起きない |
| 若手ニーズとの乖離 | 上司世代が設計した研修が若手に響かない | 参加率・満足度が低い |
| コンテンツの陳腐化 | 数年前のスライドを使い回す企業多い | DX時代に即していない |
| 中小・中堅向けの選択肢が少ない | 大手向け高単価サービスが多い | 中堅企業の研修外注ニーズが満たされない |
| 内製人材の育成コスト | 社内講師の育成に時間・コスト | 外注が続く悪循環 |

#### B2C（個人向け学習）の課題

| 課題 | 現状 | 生じる問題 |
|------|------|-----------|
| 学習継続率の低さ | 完了率10〜15% | 投資対効果が出ない、ユーザー離脱 |
| 何を学ぶかわからない | コンテンツが多すぎて選べない | 学習開始のハードルが高い |
| 孤独な学習 | 一人でコンテンツを消化するだけ | モチベーション維持が難しい |
| 成果が見えにくい | スキルが付いたかどうか分からない | 投資継続の意思決定が難しい |
| 企業研修との分断 | 会社の研修と自己学習が連携していない | 重複・無駄が多い |
| 価格感度が高い | 月1万円超は継続率が下がる | 高品質サービスのマネタイズが難しい |

---

### 3.2 UNLIDが入り込めるホワイトスペース

#### ホワイトスペース1：「中堅企業×若手定着特化」研修

**現状のギャップ**

- 大手研修会社は大企業向けに最適化されており、中堅企業向けにはコスト・設計ともに過剰
- 中小向け格安パッケージは汎用的すぎて「若手定着」「カルチャー醸成」に特化していない

**UNLIDの機会**

- 従業員100〜500名の中堅企業に対し、「若手離職を防ぐ研修プログラム」を適正価格で提供
- カスタマイズ性と継続フォローアップを武器に差別化

---

#### ホワイトスペース2：「B2B研修 × B2C自己学習」の連携サービス

**現状のギャップ**

- 企業研修と個人学習は完全に分断されており、連携したサービスはほぼ存在しない
- 社員が自主的に使っているツールと会社の研修が全く別物

**UNLIDの機会**

- 企業が発注する研修と、個人が自主的に利用するスキルアップサービスを一体化
- 「会社で研修を受け、個人でも継続学習できるプラットフォーム」
- B2Bで入口を作り、B2Cで個人課金に転換する二重収益モデル

---

#### ホワイトスペース3：「行動変容・継続学習」にフォーカスした設計

**現状のギャップ**

- コンテンツ提供型サービスは飽和。「情報」は差別化にならない
- 「継続できる仕組み・コミュニティ・成果実感」を提供するサービスは少ない

**UNLIDの機会**

- コーチング・メンタリング・ピアラーニング等の「人との関わり」を組み込む
- 学習後のアウトプット・実践・フィードバックサイクルを設計
- 「受けた研修が行動変容につながった」というROIを可視化して提供

---

#### ホワイトスペース4：「DX時代の若手向け実践的スキル」

**現状のギャップ**

- ChatGPT・AI活用、データ分析等の実践的スキルを「業務に使えるレベル」まで教えるサービスが少ない
- 大学教育・既存研修は理論偏重で実務で使いにくい

**UNLIDの機会**

- 実務直結型のDX・業務効率化スキル研修を若手向けに提供
- 企業研修としても、個人学習としても需要がある「両利き」コンテンツ

## 4. 市場規模サマリー（TAM/SAM/SOM）

### 定義

| 指標 | 意味 |
|------|------|
| **TAM**（Total Addressable Market） | 理論的に獲得可能な最大市場規模 |
| **SAM**（Serviceable Addressable Market） | UNLIDが実際にアクセスできる市場規模 |
| **SOM**（Serviceable Obtainable Market） | 現実的に獲得可能な市場規模（3〜5年以内） |
