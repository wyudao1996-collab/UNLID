# 競合分析：B2B研修外注・B2Cスキルアップ市場

## 概要

UNLID事業（企業向け研修外注サービス × 若手向けスキルアップサービス）の競合環境を分析する。
B2B（法人研修）とB2C（個人スキルアップ）の両市場を対象とし、UNLIDが取るべき差別化ポジションを明確にする。

作成日：2026年3月
バージョン：1.0

---

## 3. 競合ポジショニングマップ

```
                        高価・カスタム
                              ↑
                              |
  グロービス(法人)  ●         |         ● RMS / 産業能率大
  識学             ●         |
                             |
  TechAcademy      ●        |
  コーチング系      ●        |
B2C寄り ←──────────────────────────────────────────── → B2B寄り
  Coursera         ●        |                          ● ラーニングエージェンシー
                             |         ● インソース（法人）
  Udemy(個人)      ●        |         ● Udemy Business
  Schoo(個人)      ●        |         ● Schoo for Business
                             |
  YouTube / SNS    ●        |
                              ↓
                        安価・汎用
```

### マップ解説

| ゾーン | 主なプレイヤー | 特徴 |
|--------|--------------|------|
| 高価×B2B | RMS、産業能率大、グロービス（法人） | 大企業向けカスタム研修、高単価・長期契約 |
| 中価格×B2B | インソース、ラーニングエージェンシー、Udemy Business | 中堅企業向け、パッケージ研修 |
| 中〜高価格×B2C | TechAcademy、ポテパンキャンプ、コーチング | 転職・スキル転換特化、個人への高付加価値提供 |
| 低価格×B2C | Udemy（個人）、Schoo、YouTube | 広く薄くコンテンツ提供、継続サポートなし |
| **空白ゾーン** | **──** | **B2B×B2Cをつなぐ中価格帯・実践カスタム領域** |

## 4. UNLIDの差別化ポイント

### 4-1. 競合が手薄なホワイトスペース

**「企業が発注する × 若手社員が本当に使いたい」の両立が弱い**

現状の競合構造を見ると、以下の分断がある：

```
B2B競合（研修会社）   →  企業の人事担当者が満足するが、若手が「使いたい」と思わない
B2C競合（個人サービス）→  若手個人が使いたいが、企業の研修予算では購入されない
```

この「企業予算で買えて、若手が自ら使いたい」という領域が市場において空白になっている。

具体的なホワイトスペース：

1. **若手の自己効力感・モチベーションを起点にした研修設計**
   - 大手研修会社：スキル伝達は得意だが「自分ごと化」が弱い
   - B2Cサービス：自己動機は高いが企業購入には繋がりにくい

2. **B2C的な体験 × B2B的な信頼性の両立**
   - UX・コンテンツの質はB2Cサービスレベルで、契約・実績はB2B水準で提供する形

3. **中堅企業（従業員100〜500名）向けのカスタム度と価格のバランス**
   - 大手研修会社：中堅企業には高すぎる・画一的すぎる
   - インソース：安いが汎用的すぎてカルチャーフィットしない
   - この中間の「セミカスタム×適正価格」ゾーンが薄い

4. **継続的な学習支援（単発研修から卒業する）**
   - 多くの研修会社は単発・点の提供で終わる
   - 継続モニタリング・定期フォローが弱い

---

### 4-2. UNLIDが取るべき独自ポジション

**「若手の成長を起点にした、企業と個人双方への価値提供」**

ポジショニング定義：

```
B2B（企業発注） × B2C（若手が主体的に使いたい） × 中価格帯カスタム
```

具体的には：

- **企業への提案**：「若手の主体性・定着率を高める、現場に根ざした研修」
- **若手への体験**：「会社の研修だけど、自分のためになる・続けられる学び」
- **テクノロジー活用**：学習進捗の可視化・継続支援により研修効果を証明

このポジションを取ることで、RMSやグロービスとの真正面衝突を避けながら、インソースやSchooが提供できない「若手の主体性×企業の信頼」を同時に訴求できる。

---

### 4-3. 競合に勝てる理由・勝てない領域

#### 勝てる領域

| 領域 | 理由 |
|------|------|
| 中堅企業（100〜500名）向け若手育成 | 大手は大企業寄り・ベンチャー系はITに偏る。このセグメントへの特化が差別化になる |
| 「若手が自ら使いたい」と思うコンテンツ設計 | 既存の研修会社はコンテンツが会社目線。若手の本音ニーズ（自己成長・市場価値）から設計する余地がある |
| B2B購買 × B2C体験の両立 | 競合は一方にしか最適化されていない |
| 継続サポート・定点観測 | 単発研修ではなく継続関与することで、競合が離脱する「研修後」の支援を独占できる |
| 離職防止・定着率向上という経営指標との接続 | 研修会社の多くは「研修の満足度」で評価が終わる。離職率・定着率への直接訴求は差別化ポイント |

#### 勝てない（現状では戦わない）領域

| 領域 | 理由 |
|------|------|
| 大企業向け大規模カスタム研修 | RMS・産業能率大などが圧倒的な実績・ブランドを持つ。初期から入り込むのは困難 |
| コース数・コンテンツ量での勝負 | Udemyには絶対に勝てない。量で戦わず、質・継続性・文脈で差別化すべき |
| IT特化エンジニア転職支援 | TechAcademy・ポテパンなどが専業で展開。スキルアップ全般でポジションを取るほうが得策 |
| 価格競争 | インソースの低価格モデルとの競合は消耗戦になる。価格ではなく価値で勝負する |

## 6. 戦略的示唆

以上の競合分析から、UNLIDが取るべき初期戦略を以下のように整理する。

### 攻めるべき市場

- **B2B：従業員100〜500名・IT/サービス業・若手定着に課題がある中堅企業**
  - 大手研修会社が力を入れないセグメント
  - 「安いだけのインソース」「高すぎるRMS」の間のスペース

- **B2C：入社1〜3年目・モヤモヤしている若手（会社経由でのリーチも含む）**
  - UdemeやSchooが提供できない「継続・コミュニティ・フィードバック」を提供

### 差別化軸

1. **若手視点の設計**：若手が「自分のためになる」と感じるコンテンツ・体験
2. **継続支援**：単発研修ではなく、学習継続・行動変容まで伴走
3. **B2B×B2Cの接続**：企業の研修投資が若手の主体的成長に繋がることを可視化
4. **離職防止KPIとの接続**：研修満足度ではなく「離職率・定着率への貢献」で評価

### 避けるべき競合

- Udemyとのコース数・価格競争
- RMS・グロービスとの大企業向けカスタム研修競争
- 純粋なITスキル転職支援（TechAcademy等との正面衝突）
