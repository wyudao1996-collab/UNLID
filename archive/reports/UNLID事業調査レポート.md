## エグゼクティブサマリー

UNLID は、**「企業向け研修外注（B2B）」と「若手個人向けスキルアップ（B2C）」を一体で提供する**、日本で唯一のデュアルモデル事業を構築する。

本調査により、以下の3点が明確になった。

| 論点 | 結論 |
|------|------|
| **市場規模** | B2B研修外注市場 約5,000億円 ＋ B2C eラーニング市場 約1,200億円。両市場とも年率5〜20%成長中 |
| **顧客の解像度** | B2B：中堅企業（100〜500名）人事担当者。B2C：入社1〜3年目のモヤモヤ若手。両者の課題は深く連動している |
| **競合の空白地帯** | 「企業が買える × 若手が使いたい」を同時に満たすサービスが市場に**存在しない** |

**3〜5年での現実的目標収益：30〜60億円**（B2B SOM 30〜50億円 ＋ B2C SOM 3〜10億円）

## 1. 顧客定義

### 1-1. B2Bターゲット：研修外注ニーズを持つ企業

#### ターゲット企業プロファイル

| 属性 | 詳細 |
|------|------|
| 規模 | 従業員 **100〜500名**の中堅企業 |
| 業種 | IT・金融・サービス業（親和性高）、製造業（一部） |
| 担当者 | 人事部・人材開発部の研修担当者（30〜50代） |
| 意思決定者 | 人事部長・経営幹部 |
| 年間研修予算 | 100万〜1,000万円規模 |
| 意思決定期間 | 1〜3ヶ月（予算サイクルは秋〜冬） |

このセグメントを優先する理由は3つある。
- 予算の意思決定が比較的速く、外注経験がある
- 大手研修会社が「大企業向けに最適化」しているため、中堅企業は現状サービスに不満を持っている
- 若手の定着課題が深刻で、訴求しやすい

#### 企業が抱える研修外注の課題

```
課題の構造：

社内リソース不足（専任担当者がいない）
    ↓
コスト合理化ニーズ（内製固定費を下げたい）
    ↓
若手・新人の定着率低下（入社3年以内離職率 推定30〜35%）
    ↓
DX・法令対応の専門領域（社内では対応不可能）
    ↓
研修外注への依存度が高まる
```

**企業が外注先に求める5条件**

1. **専門性と実績** — 業界・テーマへの深い知識、導入事例
2. **カスタマイズ対応** — 自社カルチャー・課題に合わせた設計
3. **効果測定** — 研修後の行動変容・成果の可視化
4. **コスト透明性** — 明確な料金体系と費用対効果の根拠
5. **継続サポート** — 単発でなく継続的な関係構築

#### 購買行動フロー

```
課題認識 → 情報収集（Web・紹介）→ 複数社比較 → 提案依頼（RFP）→ 契約
```

### 1-2. B2Cターゲット：若手社員・若手ワーカー

#### ターゲット個人プロファイル

| 属性 | 詳細 |
|------|------|
| 年齢 | **22〜32歳** |
| 状況 | 新卒入社1〜5年目 / 第二新卒 / 転職検討中 |
| 職種 | 営業・事務・IT・サービス業など |
| 年収 | 250〜450万円 |
| 学習費支出 | 月額3,000〜10,000円が許容ライン |

#### 若手が抱えるリアルな課題

**スキル・キャリア面**

- 今の仕事が将来も通用するか分からない（将来不安）
- 何を学べばいいか、何が市場価値になるか不明
- 日々の業務が「こなし作業」で成長実感がない

**職場・人間関係面**

- 上司が忙しく指導してもらえない（フィードバック不足）
- テレワーク化で同僚とのつながりが薄い（孤立感）
- 上司の価値観・働き方と合わない（世代間ギャップ）

#### 表層ニーズ vs 深層ニーズ

| 表層ニーズ（言葉にすること） | 深層ニーズ（本当に求めていること） |
|------|------|
| スキルを身につけたい | **自信を持って仕事をしたい** |
| 資格を取りたい | **市場価値を高めて選択肢を広げたい** |
| 副業・転職したい | **今の環境に依存しない自立** |
| 給料を上げたい | **努力が正当に評価される環境** |
| コミュニケーション力を上げたい | **職場で認められたい・孤独を解消したい** |

> **示唆**：若手はスキルを求めているのではなく、**「自信と選択肢」** を求めている。これがサービス設計の根幹になる。

### 1-3. B2BとB2Cの接続構造

UNLIDが最も差別化できる構造は、B2BとB2Cを**循環モデル**として設計することにある。

```
      企業（B2B）
         ↓  研修を発注
       UNLID
         ↓  研修を提供
      若手社員（B2C）
         ↓  スキルアップ・キャリア実感
      企業（B2B）
         ↑  人材定着・生産性向上 → 継続発注
```

- **B2Bへの訴求**：「若手の離職を防ぎ、即戦力化を支援する研修」
- **B2Cへの訴求**：「会社が変わっても通用するスキルと自信」

この循環が成立すれば、B2B顧客獲得がB2C ユーザー獲得を促進し、B2Cでの実績がB2B提案の根拠になる**二重のネットワーク効果**が生まれる。

## 2. 市場調査

### 2-1. B2B 企業研修市場

#### 市場規模（国内推定）

| 区分 | 推定規模 | 備考 |
|------|----------|------|
| 企業向け人材育成費全体 | 約1兆2,000億〜1兆5,000億円／年 | 企業人件費の推定1〜2%相当 |
| うち外注・アウトソーシング | **約3,000億〜5,000億円／年** | 全体の25〜35%が外注化 |
| 研修サービス専業市場 | 約2,000億〜3,500億円／年 | eラーニング含む |

> 参考：矢野経済研究所「人材育成・研修サービス市場」、リクルートワークス研究所各種報告（概算・推計）

#### 市場成長率とトレンド

| 期間 | 成長率（推定） | 主な要因 |
|------|--------------|---------|
| 2020〜2022年 | ▲5〜10% | コロナ禍による対面研修停止 |
| 2022〜2024年 | +8〜12%／年 | オンライン研修普及・リスキリング需要 |
| 2025〜2027年（予測） | **+5〜8%／年** | DX人材育成・法令対応研修の継続需要 |

#### 注目する4つのトレンド

**① オンライン・ハイブリッド研修の定着**
コロナ禍を経て、企業研修の推定50〜60%がオンラインまたはハイブリッド形式へ移行。地方・多拠点企業でのアクセシビリティが向上した。

**② リスキリング関連予算の拡大**
政府の「リスキリング支援」施策（2022年〜）により企業の研修予算が増加。DX・AI活用スキル研修は推定年率20%超の伸び。

**③ 研修のマイクロラーニング化**
長時間集合研修から短尺・反復型へ。スマートフォン対応コンテンツの需要が増加。

**④ 研修効果測定（ROI可視化）への強いニーズ**
「やりました」で終わる研修から「行動変容の証明」へのシフトが起きており、ラーニングアナリティクスの需要が急増。

#### 外注化が進む5つのドライバー

1. **人事部門の人員不足**：中堅企業の研修担当は1〜3名（兼任）が大半
2. **コスト合理化**：内製固定費（年収500〜700万円/人）vs. 外注スポット費用の比較
3. **テーマの多様化・高度化**：DX・AI・グローバルなど社内対応が困難
4. **若手定着課題の深刻化**：新卒3年以内離職率 推定30〜35%（厚労省データ参考）
5. **法令対応**：ハラスメント対策・育児介護休業法改正等の継続的なアップデート需要

### 2-2. B2C 若手スキルアップ市場

#### 市場規模（国内推定）

| 市場区分 | 推定規模 | 成長率 |
|---------|----------|--------|
| 国内eラーニング市場全体（法人＋個人） | 約3,000億〜4,000億円／年 | +8〜12%／年 |
| **個人向けオンライン学習** | **約800億〜1,200億円／年** | **+15〜20%／年** |
| オンライン動画学習（Udemy等） | 約300億〜500億円／年 | +20〜30%／年 |
| キャリアコーチング・メンタリング | 約100億〜200億円／年 | +15〜20%／年 |

> 参考：矢野経済研究所「eラーニング市場規模」、ICT総研各種調査（概算・推計）

#### 若手の学習実態データ（推定）

**意欲と行動のギャップ**

| 指標 | データ（推定） |
|------|--------------|
| 「スキルアップしたい」と思う社会人比率 | 約70〜80% |
| 実際に学習行動に移している比率 | **約30〜40%（意欲の半分以下）** |
| 月3,000円以上支出する20〜30代 | 約20〜25% |
| オンラインコースの完了率 | **約10〜15%（業界共通の課題）** |

**費用支出の分布（若手社会人）**

| 月額支出 | 比率（推定） | 主な支出先 |
|---------|------------|-----------|
| 0円 | 約40〜45% | — |
| 〜1,000円 | 約15% | 書籍、無料アプリ |
| 1,001〜3,000円 | 約20% | サブスク型eラーニング |
| 3,001〜10,000円 | 約15% | コーチング、有料コース |
| 10,001円以上 | 約5〜10% | スクール、資格講座 |

#### B2C市場の成長ドライバー

1. **AI・DXによるスキル陳腐化不安**：ChatGPT等の普及で「自分の仕事がなくなるかも」という不安が顕在化
2. **転職市場の活性化**：「終身雇用神話の崩壊」が若手層に浸透し、スキルポータビリティへの意識が高まっている
3. **副業・フリーランス市場の拡大**：副業解禁の流れで本業以外の収入源を求める若手が増加（フリーランス人口 推定300万人）
4. **学習コンテンツのコモディティ化**：情報は「タダ」に近い時代になり、「行動変容・習慣化・コミュニティ」に価値がシフト

### 2-3. 既存市場の非効率（アンメットニーズ）

#### B2B（企業研修）の課題

| 課題 | 現状 | 生じる問題 |
|------|------|-----------|
| 研修効果の不透明さ | 「やりました」で終わる研修が多い | 予算承認が取りにくい、継続発注されない |
| 単発・点の施策 | 年1回の集合研修で終了 | 学習定着率が低く行動変容が起きない |
| **若手ニーズとの乖離** | 上司世代が設計した研修が若手に響かない | 参加率・満足度が低い |
| 中堅向け選択肢不足 | 大手向け高単価・汎用格安の二極化 | 中堅企業のニーズが満たされない |

#### B2C（個人向け学習）の課題

| 課題 | 現状 | 生じる問題 |
|------|------|-----------|
| 学習継続率の低さ | 完了率10〜15% | 投資対効果が出ない |
| 何を学ぶか分からない | コンテンツが多すぎて選べない | 学習開始のハードルが高い |
| 孤独な学習 | 一人でコンテンツを消化するだけ | モチベーション維持が難しい |
| **企業研修との分断** | 会社の研修と自己学習が連携していない | 重複・無駄が多い |

## 4. ホワイトスペースとUNLIDの戦略ポジション

### 4-1. 競合構造の本質的な分断

現在の競合が解消できていない構造的な矛盾が1つある。

```
B2B競合（研修会社）    → 企業の人事担当者が満足するが、若手が「使いたい」と思わない
B2C競合（個人サービス）→ 若手個人が使いたいが、企業の研修予算では購入されない
```

**「企業が買えて、若手が自ら使いたい」を同時に満たすサービスが、日本市場に存在しない。**

### 4-2. UNLIDの4つのホワイトスペース

#### ホワイトスペース① 中堅企業×若手定着特化

- 大手研修会社は大企業向けに最適化（コスト・設計ともに中堅には過剰）
- 格安パッケージは汎用的すぎて「若手定着」「カルチャー醸成」に特化していない
- **→ 従業員100〜500名の中堅企業に、若手離職を防ぐ研修を適正価格で提供**

#### ホワイトスペース② B2B研修×B2C自己学習の連携

- 企業研修と個人学習は完全に分断されており、連携したサービスはほぼ存在しない
- **→ 企業が発注する研修と、個人が自主的に利用するスキルアップを一体化**

#### ホワイトスペース③ 行動変容・継続学習へのフォーカス

- コンテンツ提供型は飽和。「継続できる仕組み・コミュニティ・成果実感」を提供するサービスが少ない
- **→ コーチング・ピアラーニング・フィードバックサイクルを組み込んだ設計**

#### ホワイトスペース④ DX時代の実践的スキル

- ChatGPT・AI活用、データ分析を「業務で使えるレベル」まで教えるサービスが不足
- **→ 実務直結型DX・業務効率化スキルを若手向けに提供（B2B・B2C双方に需要）**

---

### 4-3. UNLIDの競合優位性整理

#### 勝てる領域

| 領域 | 理由 |
|------|------|
| 中堅企業（100〜500名）向け若手育成 | 大手は大企業寄り・ベンチャーはITに偏る。このセグメント特化が差別化 |
| 「若手が自ら使いたい」コンテンツ設計 | 既存研修会社は会社目線。若手の本音ニーズ（自己成長・市場価値）から設計する余地がある |
| B2B購買 × B2C体験の両立 | 競合は一方にしか最適化されていない |
| 継続サポート・定点観測 | 単発研修ではなく継続関与。「研修後」の支援を独占できる |
| 離職防止・定着率向上という経営指標との接続 | 研修会社の多くは「研修の満足度」で評価終了。離職率・定着率への直接訴求は差別化 |

#### 勝てない（戦わない）領域

| 領域 | 理由 |
|------|------|
| 大企業向け大規模カスタム研修 | RMS・産業能率大などが圧倒的な実績・ブランドを保有 |
| コース数・コンテンツ量での勝負 | Udemyには絶対に勝てない。量ではなく質・継続性・文脈で差別化 |
| IT特化エンジニア転職支援 | TechAcademy・ポテパンが専業展開 |
| 価格競争 | インソースとの消耗戦は避ける。価格ではなく価値で勝負 |

## 5. 市場規模サマリー（TAM/SAM/SOM）

### 定義

| 指標 | 意味 |
|------|------|
| **TAM**（Total Addressable Market） | 理論的に獲得可能な最大市場規模 |
| **SAM**（Serviceable Addressable Market） | UNLIDが実際にアクセスできる市場規模 |
| **SOM**（Serviceable Obtainable Market） | 現実的に獲得可能な規模（3〜5年以内） |

## 6. 戦略的提言

### 6-1. 攻めるべき市場と差別化軸

**フォーカスすべき市場**

```
B2B：従業員100〜500名・IT/サービス業・若手定着に課題がある中堅企業の人事担当者
B2C：入社1〜3年目・モヤモヤしている若手（会社経由でのリーチも含む）
```

**差別化の4軸**

1. **若手視点の設計**：若手が「自分のためになる」と感じるコンテンツ・体験の提供
2. **継続支援**：単発研修ではなく、学習継続・行動変容まで伴走する仕組み
3. **B2B×B2Cの接続**：企業の研修投資が若手の主体的成長に繋がることを可視化
4. **離職防止KPIとの接続**：研修満足度ではなく「離職率・定着率への貢献」で評価

---

### 6-2. 避けるべき競合シナリオ

| 避けるべき競合 | 理由 |
|--------------|------|
| Udemyとのコース数・価格競争 | 資本力の差で消耗戦になる |
| RMS・グロービスとの大企業向けカスタム研修競争 | 実績・ブランド力で太刀打ちできない |
| TechAcademy等とのITスキル転職支援 | 特化型との正面衝突は得策でない |
| インソースとの価格競争 | 低価格モデルとの戦いは収益を圧迫する |

---

### 6-3. UNLIDのユニークな価値提案（まとめ）

> **「企業が買える研修を、若手が本当に使いたいと思う体験で届ける唯一のサービス」**

これは単なるスローガンではなく、競合分析から導き出された**市場の空白**に基づいている。

- B2B競合（研修会社）が「企業が買う」ことに最適化される一方、若手の体験は二の次
- B2C競合（個人サービス）が「若手が使いたい」ことに最適化される一方、企業予算での購入は設計されていない

UNLIDはこの**両側のニーズを同時に満たす設計**を持ち、競合が意図的に手を抜いているわけでもない「構造的な空白」を狙う。

## 付記：調査上の注意事項

- 本レポートの市場規模数値はすべて**推定・概算**であり、公開情報・業界知見を元に合理的に算出したものである
- 競合他社の価格帯・サービス詳細は公開情報をもとに作成。「推定」と記載した箇所は一般的な市場知識をもとにした推計値であり、各社公式情報で確認を要する
- 市場規模の定義・調査手法により数値は大きく異なる場合がある
- 事業計画への転用時は、最新の統計データ（矢野経済研究所・ICT総研・厚生労働省等）による検証を推奨する
- SOM試算は保守的シナリオを想定しており、PMF（製品市場適合）達成後は大幅に上振れる可能性がある
