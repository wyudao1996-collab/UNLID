# 顧客定義：B2B研修外注ニーズ と B2C若手実態

## 概要

UNLID事業における顧客セグメントを、B2B（企業向け）とB2C（個人向け）の2軸で定義する。

---

## B2B セグメント：研修外注ニーズを持つ企業

### ターゲット企業像

| 属性 | 詳細 |
|------|------|
| 規模 | 従業員50〜1,000名規模の中堅企業 |
| 業種 | IT・製造・金融・サービス業など |
| 担当者 | 人事部・人材開発部・研修担当者（30〜50代） |
| 意思決定者 | 人事部長・総務部長・経営幹部 |

### 外注ニーズの背景

#### 1. 社内リソース不足
- 専任の研修担当者がいない（兼任が多い）
- 研修コンテンツ作成のノウハウ・時間が不足
- 外部の専門知識・最新トレンドへのアクセスが困難

#### 2. コスト効率化ニーズ
- 内製化にかかる固定費（人件費・ツール費）を削減したい
- 研修効果が測定できず、ROIが不透明
- スポット的な研修ニーズに対して固定費をかけたくない

#### 3. 若手・新人育成の課題
- 入社後3年以内の離職率が高い
- OJTだけでは体系的なスキル習得が難しい
- 世代間ギャップ（上司が若手に教えられない領域がある）

#### 4. 変化への対応
- DX推進に伴うリスキリング需要
- ハラスメント対策・コンプライアンス研修の義務化対応
- 多様性（DE&I）教育の必要性

### 企業が研修外注先に求めるもの

1. **専門性と実績** — 業界・テーマへの深い知識、導入事例
2. **カスタマイズ対応** — 自社の課題・カルチャーに合わせた設計
3. **効果測定** — 研修後の変化・成果の可視化
4. **コスト透明性** — 明確な料金体系、費用対効果の説明
5. **継続サポート** — 単発でなく継続的な関係構築

### 購買行動パターン

```
課題認識 → 情報収集（Web検索・紹介） → 複数社比較 → 提案依頼（RFP） → 契約
```

- 意思決定までのリードタイム：1〜3ヶ月
- 予算サイクル：年度予算（4月始まり）に合わせた秋〜冬の検討が多い
- 決め手：担当者との信頼関係・提案内容の具体性

## B2C セグメント：若手社員・若手ワーカーの実態

### ターゲット個人像

| 属性 | 詳細 |
|------|------|
| 年齢 | 22〜32歳 |
| 属性 | 新卒入社1〜5年目 / 第二新卒 / 転職検討中 |
| 職種 | 営業・事務・IT・製造・サービス業など |
| 収入 | 年収250〜450万円 |

### 若手が抱えるリアルな課題

#### キャリア・スキル面
- **将来への不安**：今の仕事が将来も通用するか分からない
- **スキルの実感なさ**：何を学べばいいか、何が市場価値になるか不明
- **成長実感の欠如**：日々の業務が「こなし作業」になっている
- **会社頼みの限界**：社内研修だけでは物足りない、でも何から始めればいい？

#### 職場・人間関係面
- **フィードバック不足**：上司が忙しく、指導してもらえない
- **孤立感**：テレワーク化で同僚とのつながりが薄い
- **世代間ギャップ**：上司の価値観・働き方と合わない

#### 学習行動の実態
- 学習意欲はあるが**継続が難しい**（三日坊主問題）
- 独学（YouTube・書籍・Udemy）は試みるが**体系性がない**
- 仕事が忙しく**学習時間の確保が難しい**（平日は1〜2時間が限界）
- **成果が見えにくい**学習は続かない

### 若手の本質的ニーズ（表層 vs 深層）

| 表層ニーズ（言葉にすること） | 深層ニーズ（本当に求めていること） |
|------|------|
| スキルを身につけたい | 自信を持って仕事をしたい |
| 資格を取りたい | 市場価値を高めて選択肢を広げたい |
| 副業・転職したい | 今の環境に依存しない自立 |
| 給料を上げたい | 努力が正当に評価される環境 |
| コミュニケーション力を上げたい | 職場で認められたい・孤独を解消したい |

### 情報収集・購買行動

- **情報源**：SNS（X/Instagram/TikTok）、YouTube、友人・先輩の口コミ
- **信頼形成**：無料コンテンツ → 無料体験 → 有料転換（時間をかけて）
- **価格感度**：月額3,000〜10,000円が許容ライン（サブスク慣れ）
- **継続条件**：成果実感・コミュニティ感・使いやすさ

## B2BとB2Cの接点（両利きの視点）

UNLIDが価値を提供できる構造：

```
企業（B2B）
  ↓ 研修を発注
UNLID
  ↓ 研修を提供
若手社員（B2C）
  ↓ スキルアップ・キャリア実感
企業（B2B）
  ↑ 人材定着・生産性向上
```

- **B2Bへの訴求**：「若手の離職を防ぎ、即戦力化を支援する研修」
- **B2Cへの訴求**：「会社が変わっても通用するスキルと自信」

## 優先顧客（初期フォーカス）

### B2B 優先ターゲット
→ **従業員100〜500名・IT/サービス業・若手育成に課題感がある人事担当者**

理由：
- 予算の意思決定が比較的速い
- 外注経験があり、提案が通りやすい
- 若手の定着課題が深刻で訴求しやすい

### B2C 優先ターゲット
→ **入社1〜3年目・なんとなくモヤモヤしている・学びたいが何から始めればいいか分からない若手**

理由：
- 課題認識が明確で行動変容しやすい
- 口コミ・紹介が広がりやすい年代
- B2Bへのリファレンス（「社員が自主的に使っている」）になりうる
